# 統計ページ構想（stats）

長い言葉のコレクションを「数字と分布」で見せる公開ページの構想メモ。
掲載候補の統計カタログと、実装・デザイン上の前提をまとめる。

- 対象: 公開側（閲覧のみ）。管理者専用機能ではない。
- 前提データ規模: 想定1万レコード程度。集計はインデックス済みカラムへの `GROUP BY` が中心で、リアルタイム集計で足りる（キャッシュ基盤は当面不要）。
- 現状スナップショット（2026-07-12 時点）: 単語 99 / 語義 101。アノテーション（ジャンル・品詞・エンティティ）付与は 23 件、特徴付与 5 行、語種付与 29 行。**アノテーション依存の統計は付与率が上がってから公開する**か、「集計対象は注釈済み N 件」と明示して載せる。

---

## 1. 全語義で成立する統計（データが常にある）

読み・表層形からの派生カラムはすべての語義に入っているため、件数が少なくても成立する。第一弾はここから選ぶ。

| 統計 | 元データ | 見せ方の案 |
|---|---|---|
| 読みの長さ分布 | `reading_length` | ヒストグラム。**波形塗りバー**（索引で不採用→統計ページ用に温存した案）の出番。看板統計 |
| 最長語ランキング | `reading_length` | トップ10 リスト。語詳細への導線 |
| モーラ数分布 | `mora_count` | ヒストグラム。文字数との散布図、「文字数とモーラ数がズレる語」（拗音・長音が多い語）ランキング |
| 頭文字・末尾文字の50音分布 | `first_char` / `last_char` | 50音表ヒートマップ（索引ページの朱ヒートを流用）。「しりとりで強い/詰む文字」的な読み物にも |
| 文字種構成 | `char_type_pattern` | 漢字のみ / ひらがな混じり / カタカナのみ / 英字混在 の構成比。純カタカナ語率など |
| 母音・韻 | `vowel_pattern` / `rhythm_pattern` | 母音出現頻度、同母音連続語、脚韻グルーピング。独自性が高い |
| 収録数の推移 | `created_at` | 累積曲線・月別登録数。コレクションの成長記録 |

## 2. アノテーション依存の統計（付与率が上がったら）

### 2.1 各マスタ単独の構成比

- **ジャンル分布**: 大分類10種の構成比が基本形。隣接リスト3階層を活かして大→中→小へドリルダウン（墨罫のネスト矩形ツリーマップ。角丸・グラデは design.md により禁止）。「語が集まっている小分類トップ10」も。
- **品詞構成比**: 名詞への極端な偏りが予想され、その偏り自体が見せられる事実。少数派（動詞・副詞・連語）は全件列挙でも成立。
- **エンティティタイプ内訳**: まず一般語 vs 固有名詞的な語の比率、次に固有名詞側の内訳（作品名・地名・定理法則名…18種）。「ことわざ」「発言」など珍しいタイプは件数＋実例を添える。
- **語種構成**: 和語（日本語）/ 漢語（中国語）/ 外来語各言語の比率。多対多を活かした**混種語率**と**語種の組み合わせ頻度**（日本語＋英語 など）が独自統計。
- **言語学的特徴ランキング**: 連濁・熟字訓・重箱読みなど20種の件数ランキング。1語義あたり平均特徴数、「特徴てんこ盛り語」ランキング。

### 2.2 派生カラムとのクロス集計（このDBならではの層）

仮説検証型の統計。アプリの趣旨「長い言葉を集めて解析する」を最も体現する。

- **ジャンル × 平均読み長**: どの分野の言葉が長いか
- **語種 × 読み長**: 外来語は長いのか、漢語複合語が最長か
- **語種 × 文字種パターン**: カタカナ語＝外来語の対応がどれくらい崩れるか（当て字・漢訳語）
- **特徴 × 語種**: 連濁・音便は和語に集中するはず、という定説の検証
- **エンティティ × ジャンル**: 作品名→芸術、定理・法則名→自然科学 の対応マトリクス

### 2.3 特徴データの中身（target / target_reading）の分析

`word_sense_features` は該当部分の文字列を持つため、件数以上のことが掘れる。

- **連濁で濁る行の頻度**: 濁音化した音がガ/ザ/ダ/バ行のどれかを集計（連濁研究の定番の切り口）
- **特徴の共起**: 同じ語義に同時に付く特徴ペア（連濁×畳語 など）
- **特徴密度**: 読み長に対する特徴数（長い語ほど連濁を含みやすいか）

### 2.4 その他

- **多義語率**: 語義2つ以上の単語の割合。同音異義語の件数
- **別表記を持つ語の割合**
- **意味文の長さ**: `meaning` の平均文字数、最長の意味を持つ語
- **注釈カバレッジ**: ジャンル・品詞等の付与率。公開統計というよりデータベースの完成度を示すメタ統計。CC BY 4.0 で公開するデータの品質指標にもなる

## 3. デザインメモ

- design.md「活字見本帖」に従う: 朱一色アクセント、角丸・影・グラデ禁止、数値ラベルは極小等幅。
- 読みの長さ分布には、索引ページで不採用にした**波形塗りバー案**（セルを件数比例で朱に塗り上辺を正弦波にする。SVG `path` を JS 生成）を充てる予定として温存済み。
- 50音ヒートマップは索引の `.kana-cell--heat`（朱の濃淡 + `color-mix`）を流用。
- グラフ描画のためのチャートライブラリは導入しない（インライン SVG + 最小限の JS で描く）。

## 4. 第一弾の推奨構成

全件にデータがあり見栄えが揃う4点:

1. 読みの長さ分布（波形塗りバー）
2. 最長語ランキング
3. 頭文字ヒートマップ（50音表）
4. 収録数の推移

アノテーション付与率が上がった段階で、ジャンル分布・語種構成・特徴ランキング、次いでクロス集計（ジャンル×読み長、特徴×語種）を追加する。
